在类中使用_property访问私有成员变量是没有问题的，使用self.property也可以，但是后者不能用在init 和 dealloc中，程序中全部使用_property这种形式会带来微小的优化，虽然在我们的小程序中体现不出来。 早期的 Objective-C 语言，类的私有成员变量是只能定义在 .h 的头文件里面的，后来可以加到.m文件中，用如下形式： @interface EverydayTVC() @property (nonatomic, assign) int page; @end

后来又允许在.m中直接添加成员变量： @implementation EverydayTVC { NSInteger _num; } 推荐用这种方法和_property访问私有成员变量的方法。 文章参考自唐巧大神的博客http://www.devtang.com，自己做个记录。
